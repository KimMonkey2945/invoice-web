# 노션 기반 견적서 공유 서비스 (invoice-web) MVP PRD

## 🎯 핵심 정보

**목적**: 발행자가 노션에 입력한 견적서를, 고객이 가입 없이 웹 링크로 확인하고 PDF로 받을 수 있게 한다.
**사용자**: 견적서를 발행하는 프리랜서·소규모 사업자와, 그 견적서를 링크로 전달받아 확인하는 고객.

## 🚶 사용자 여정

```
1. 견적서 확인 페이지 (발행자가 전달한 고유 링크로 접속)
   ↓ 견적서 ID 유효성 체크

   [유효한 ID] → 노션에서 견적서 조회 → 견적서 내용 화면 표시
   [없는/만료 ID] → 견적서 오류 페이지 (안내 문구 표시)
   ↓

2. 견적서 내용 확인 (항목·금액·유효기간 열람)
   ↓ "PDF 다운로드" 버튼 클릭

3. PDF 생성 및 다운로드
   ↓ 완료

4. [완료] → 견적서 재열람 / 페이지 종료

* 홈(랜딩) 페이지는 서비스 소개 전용이며, 견적서 열람은 항상 고유 링크로만 진입한다.
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID         | 기능명              | 설명                                                                       | MVP 필수 이유                             | 관련 페이지        |
| ---------- | ------------------- | -------------------------------------------------------------------------- | ----------------------------------------- | ------------------ |
| **[F001]** | 견적서 데이터 조회  | 노션 API로 견적서 ID에 해당하는 단건 데이터(견적 항목·금액·정보) 페칭      | 견적서 내용을 보여주기 위한 데이터 공급원 | 견적서 확인 페이지 |
| **[F002]** | 견적서 웹 상세 표시 | 발행자/고객 정보, 견적 항목 표, 합계 금액, 발행일·유효기간을 화면에 렌더링 | 고객이 견적 내용을 확인하는 핵심 가치     | 견적서 확인 페이지 |
| **[F003]** | 견적서 PDF 다운로드 | 화면에 표시된 견적서를 동일 레이아웃의 PDF 파일로 생성·다운로드            | 고객이 견적서를 보관/공유하는 핵심 산출물 | 견적서 확인 페이지 |

### 2. MVP 필수 지원 기능

| ID         | 기능명                        | 설명                                                                   | MVP 필수 이유                                    | 관련 페이지                            |
| ---------- | ----------------------------- | ---------------------------------------------------------------------- | ------------------------------------------------ | -------------------------------------- |
| **[F004]** | 링크 기반 접근 및 유효성 처리 | 견적서 고유 ID(토큰) 기반 접근, 없는/만료 견적서에 대한 오류 안내 처리 | 가입 없는 안전한 견적서 공유의 최소 접근 제어    | 견적서 확인 페이지, 견적서 오류 페이지 |
| **[F005]** | 서비스 소개                   | 서비스 개요와 "견적서는 발행자가 보낸 링크로 확인" 안내 표시           | 직접 방문자에게 이용 방법을 안내하는 최소 진입점 | 홈 페이지                              |

### 3. MVP 이후 기능 (제외)

- 발행자용 견적서 작성/편집 화면 (입력은 노션에서 직접 수행)
- 회원가입/로그인 기반 인증 및 발행자 대시보드
- 견적서 상태 변경 알림, 이메일 발송
- 결제·전자서명·견적 승인 워크플로우
- 다국어/통화 변환, 고급 검색 및 필터링

## 📱 메뉴 구조

```
📱 견적서 공유 서비스 내비게이션
├── 🏠 홈 (랜딩)
│   └── 기능: F005 (서비스 소개)
└── 📄 견적서 확인 (고유 링크로만 진입)
    └── 기능: F001, F002, F003, F004 (조회 / 표시 / PDF / 링크 검증)

🔀 조건부 화면 (메뉴 미노출, 자동 분기)
└── ⚠️ 견적서 오류 (없는/만료 견적서 접근 시 자동 이동)
    └── 기능: F004 (유효성 처리)

* 본 서비스는 고객 로그인이 없으므로 사용자별 메뉴/공통 인증 메뉴는 제공하지 않는다.
```

## 📄 페이지별 상세 기능

### 홈 페이지

> **구현 기능:** `F005` | **메뉴 위치:** 내비게이션 홈

| 항목            | 내용                                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 서비스 소개 랜딩 페이지 (직접 방문자 안내)                                                                                 |
| **진입 경로**   | 사이트 루트 직접 접속                                                                                                      |
| **사용자 행동** | 서비스 개요를 읽고, 견적서는 발행자가 보낸 링크로 확인해야 함을 인지                                                       |
| **주요 기능**   | • 서비스 한 줄 소개 및 이용 방법 안내<br>• "견적서는 전달받은 링크로 확인하세요" 안내 문구<br>• 견적서 예시/특징 간단 소개 |
| **다음 이동**   | 별도 내비게이션 없음 (견적서 열람은 외부에서 받은 고유 링크로 진입)                                                        |

---

### 견적서 확인 페이지

> **구현 기능:** `F001`, `F002`, `F003`, `F004` | **인증:** 없음 (고유 링크/ID 기반 접근)

| 항목            | 내용                                                                                                                                                                                      |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 고객이 견적서 내용을 확인하고 PDF로 받는 핵심 페이지                                                                                                                                      |
| **진입 경로**   | 발행자가 전달한 견적서 고유 ID가 포함된 링크로 접속                                                                                                                                       |
| **사용자 행동** | 견적 항목·금액·유효기간을 확인하고 PDF 다운로드 버튼을 누름                                                                                                                               |
| **주요 기능**   | • 노션에서 해당 견적서 데이터 조회 (F001)<br>• 발행자/고객 정보, 견적 항목 표, 합계 금액, 발행일·유효기간 표시 (F002)<br>• 견적서 ID 유효성 검증 (F004)<br>• **PDF 다운로드** 버튼 (F003) |
| **다음 이동**   | ID 유효 → 견적서 표시, ID 없음/만료 → 견적서 오류 페이지                                                                                                                                  |

---

### 견적서 오류 페이지

> **구현 기능:** `F004` | **진입:** 자동 분기 (메뉴 미노출)

| 항목            | 내용                                                                                                    |
| --------------- | ------------------------------------------------------------------------------------------------------- |
| **역할**        | 존재하지 않거나 만료된 견적서 접근 시 안내 전용 페이지                                                  |
| **진입 경로**   | 견적서 확인 페이지에서 ID가 없거나 만료로 판정될 때 자동 이동                                           |
| **사용자 행동** | 견적서를 볼 수 없는 이유를 확인하고, 발행자에게 문의하도록 안내받음                                     |
| **주요 기능**   | • "견적서를 찾을 수 없거나 유효기간이 지났습니다" 안내<br>• 발행자 문의 안내 문구<br>• 홈으로 이동 링크 |
| **다음 이동**   | 홈 페이지                                                                                               |

## 🗄️ 데이터 모델

> 실제 저장소는 노션 데이터베이스이며, 아래 모델은 노션 속성(property)에 대응한다.

### Quotation (견적서)

| 필드            | 설명                                          | 타입/관계 |
| --------------- | --------------------------------------------- | --------- |
| id              | 견적서 고유 식별자 (노션 page id / 링크 토큰) | UUID      |
| quotationNumber | 견적서 번호 (예: INV-2026-001)                | title     |
| issuerName      | 발행자(업체/개인) 이름                        | rich_text |
| clientName      | 고객 이름/회사                                | rich_text |
| issueDate       | 발행일                                        | date      |
| validUntil      | 유효기간(만료일)                              | date      |
| status          | 견적서 상태 (유효/만료)                       | select    |
| note            | 비고/안내 문구                                | rich_text |

### LineItem (견적 항목)

| 필드        | 설명               | 타입/관계      |
| ----------- | ------------------ | -------------- |
| id          | 항목 고유 식별자   | UUID           |
| quotationId | 소속 견적서        | → Quotation.id |
| name        | 항목명/품목 설명   | title          |
| quantity    | 수량               | number         |
| unitPrice   | 단가               | number         |
| amount      | 금액 (수량 × 단가) | number         |
